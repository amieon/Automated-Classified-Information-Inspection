# 🔐 自动涉密信息检测系统


## 📌 项目简介

本项目基于 Python + FastAPI 构建，实现了一个**多数据源自动涉密信息检测系统**。系统支持对网页、数据库、文件以及图片中的文本内容进行扫描，通过关键字匹配与正则表达式分析，自动识别潜在的涉密信息，并生成结构化检测报告。

系统具备良好的模块化设计，支持扩展不同类型的数据检测任务，并提供简单易用的 Web 用户界面，方便用户输入检测目标并查看结果。

## 🎯 实现功能

本系统完整实现了五大检测模块：

---

### 🌐 1. 网页涉密信息检测

* 对目标网站进行爬取（如：https://bm.yangyq.net/）
* 遍历站内所有网页（忽略外链）
* 提取网页文本内容进行检测
* 支持检测网页中的文档链接（可选）

✅ 输出内容：

* 检测网页总数
* 涉密网页数量
* 涉密信息出现位置（具体行号）

---

### 🗄️ 2. 数据库涉密信息检测

* 支持 MySQL / MariaDB 数据库
* 遍历所有表及字段
* 检查每条记录中的文本数据

✅ 输出内容：

* 数据库表数量
* 每个表的数据条数
* 涉密数据条数
* 涉密字段所在位置（表名 + 字段名）

---

### 📂 3. 文件涉密信息检测

支持文件类型：

* Word（.doc / .docx）
* Excel（.xls / .xlsx）
* PowerPoint（.ppt / .pptx）
* PDF（.pdf）
* 文本（.txt）

功能特点：

* 递归遍历目录
* 自动识别文件类型（解决后缀不匹配问题）
* 支持隐藏文件检测
* 尝试处理加密文件（部分支持）

✅ 输出内容：

* 文件总数及分类统计
* 涉密文件数量
* 涉密信息所在文件及行号

---

### 🖼️ 4. 图片涉密信息检测

* 遍历所有图片文件（jpg/png等）
* 使用本地 OCR 技术提取文字（不使用远程API）
* 对识别结果进行涉密检测

✅ 输出内容：

* 图片总数及类型统计
* 涉密图片数量
* 涉密图片路径及文件名

---

### 🎵 5. 音频涉密信息检测

* 遍历所有音频文件（mp3/wav等）
* 使用本地神经网络whisper提取文字（不使用远程API）
* 对识别结果进行涉密检测

✅ 输出内容：

* 音频总数及类型统计
* 涉密音频数量
* 涉密音频路径及文件名

---

### 🔍 6. 涉密信息检测策略

系统采用关键字 + 正则表达式的方式进行检测：

#### 关键词：

```
秘密, 机密, 绝密, 内部, 涉密, 军事秘密, 国家秘密, 商业秘密, 保密, 密级, 不予公开
```
#### 关键词可以自行更改：
位于checker/keywords.py中，默认如下
```
KEYWORDS = [
            "秘密", "机密", "绝密", "内部", "涉密",
            "军事秘密", "国家秘密", "商业秘密",
            "保密", "密级", "不予公开"
        ]
```
#### 正则增强：

* 支持空格/任意符号干扰匹配（如：绝 密、机-密、机123密）
* 干扰强度可以自己改（在regex_leak_detector.py中），默认为3个字符
* 支持模糊匹配，提高检测鲁棒性

## 🧱 系统架构

```
├── main.py                # 系统入口（FastAPI）
├── checkers/             # 各类检测模块
│   ├── web_checker.py
│   ├── file_checker.py
│   ├── image_checker.py
│   ├── audio_checker.py
│   └── db_checker.py
├── utils/                # 核心工具模块
│   ├── crawler.py
│   ├── leak_detector.py
│   ├── regex_leak_detector.py
│   ├── office_parser.py
│   └── report_generator.py
├── templates/            # 前端页面
│   └── index.html
└── static/               # 静态资源
    ├── bootstrap.bundle.min.js
    ├── bootstrap.min.css
    └── download.py
```

### ✨ 架构特点

* 插件化设计（各检测模块独立）
* 动态加载检测模块（高扩展性）
* 前后端分离（FastAPI + HTML）
* 工具层与业务层解耦

## 📊 报告生成

系统自动生成检测报告，包含：

* 总体统计信息（数量、比例）
* 涉密数据详细位置
* 分类结果（网页 / 文件 / 图片 / 数据库）
* 概述性总结（检测情况说明）

📌 特点：

* 结构清晰
* 信息完整
* 无需人工整理
